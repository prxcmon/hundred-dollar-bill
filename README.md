# Hundred Dollar Bill (C-Note)

## Introduction
This repository contains my personal learning guide of C programming language based on several books, projects, and slides, including:
- [“Hacking: The Art of Exploitation (2nd Edition), by Jon Erickson”](https://www.amazon.com/Hacking-Art-Exploitation-Jon-Erickson/dp/1593271441);
- ["Essential C, by Nick Parlante"](https://www.amazon.com/Essential-C-Nick-Parlante/dp/1521314233)
- ["Le Langage C, par Lucas-84, Taurre, et informaticienzero — Zeste de Savoir"](https://zestedesavoir.com/tutoriels/755/le-langage-c-1/)
- ["VRDP Training Labs (C and Python)"](https://github.com/interruptlabs/VRDP-C-Python)
- and other sources as well, which I will add it later.
