## Programming Starters

Before starting this project, you will be making your first program. Rewrite (not copy-and-paste) this code, then run it into one of your favorite compilers.
```c
# include <stdio.h>

int main()
{
    printf("ID: Halo, dunia!\n");
    printf("EN: Hello, world!\n");
    printf("FR: Bonjour, tout le monde!\n");
    return 0;
}
```

Save it as a new code `bonjour.c`, then compile and run the object to display the string.
```bash
$ gcc bonjour.c -o objet
$ ./objet
```
